## Objective
Verify an FLB recovery point copied (via Backup Copy Job) into an immutable cloud repository
(Amazon S3, per the Xray body text) can be recovered to a CIFS share matching source content, with
the Object Lock remaining intact.

- Source: an existing FLB backup object on Onboard
- Repository: **Amazon_Immutable** (`BACKUP_REPOSITORY-10`) — added 2026-07-08
- Job build: **R4d** — `backup_copy_job.template.json` (`hvType:"VMWARE"` fixed,
  `targetStorageVid=BACKUP_REPOSITORY-10`) + `retentionPolicy.retentionMode="RULESET"`
- Job name: `AUTO_FLB_NJM-123181`

- [ ] Source `BACKUP_OBJECT` present; `Amazon_Immutable` repo online.
