## Objective
Verify an FLB recovery point copied (via Backup Copy Job) into an immutable Azure Blob repository
can be recovered via a browser ZIP download, with content matching source and the Azure
immutability lock intact.

## Fixtures used
- Source: an existing FLB backup object on Onboard (from a prior `AUTO_FLB_*` run)
- Repository: **Azure_Immutable** (`BACKUP_REPOSITORY-12`) — added 2026-07-08
- Job build: **R4d** — `backup_copy_job.template.json` (`hvType:"VMWARE"` fixed,
  `targetStorageVid=BACKUP_REPOSITORY-12`) + `retentionPolicy.retentionMode="RULESET"`
- Job name: `AUTO_FLB_NJM-123179`

- [ ] Source `BACKUP_OBJECT` present; `Azure_Immutable` repo online.
