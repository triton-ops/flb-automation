## Objective
Verify an FLB recovery point stored in an immutable Backblaze B2 repository can be recovered to a
CIFS share, matching source content, with the Backblaze Object Lock remaining intact.

- Source: windows-src **PM-3** (a small fixture folder, e.g. `ft_pdf`)
- Repository: **BlackBlaze_Immutable** (`BACKUP_REPOSITORY-13`) — added 2026-07-08

- Job name: `AUTO_FLB_NJM-123182`

### Step 1 — Introspect — `mcp__nbr__describe_method nbr-84 JobManagement saveJob`.
### Step 2 — Build — R4c: `sourceVid=PM-3`, select fixture folder, `retentionPolicy.retentionMode="RULESET"`,
